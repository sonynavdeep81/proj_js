1. Install nodejs
sudo apt install nodejs
2. Install npm (node package manager) --> used to install tools
sudo apt install npm
3.Install live-server
sudo npm install live-server -g (means globally)
How to use live-server
1. Create an index.html file 
2. Include js file in install
3. Run live-server in the terminal
4. Open browser console.
5. When you save changes in vscode, automatically the changes will be reflected in chrome
